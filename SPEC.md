# Personal Website Specification & Design Doc

## 1. Overview
**Concept:** A minimalistic, linear one-page portfolio for a Research Scientist at a major tech company, emphasizing high-quality design and engineering prowess.

**Structure:**
- **Header/Hero:** Profile photo, name, concise introduction.
- **Contact:** Minimal links (Email, X, GitHub, Scholar).
- **News:** Chronological updates (short & punchy).
- **Selected Publications:** Highlighted papers with links/abstracts.
- **Blog:** List of recent posts (links to detail views).

## 2. Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** Shadcn UI (Primary), Lucide React (Icons)
- **Animation:** Motion Primitives, Motion (Framer Motion)
- **Content:** MDX (for blog posts)

## 3. Design Guidelines
**Core Aesthetic:**
- **Minimalist & Clean:** generous whitespace, disciplined typography.
- **Linear Flow:** Single-page scrolling experience for primary content.
- **Refined:** Subtle interactions and smooth animations to demonstrate frontend skill without clutter.

**UX Goals:**
- Information density should be low but high-signal.
- Navigation should be effortless (sticky nav or clear section markers).
- Dark/Light mode support (consistent with system preferences).
